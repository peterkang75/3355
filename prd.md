## 1. 회원 관리

### 1-1. 로그인/회원가입
- 전화번호 기반 로그인/회원가입
- 회원가입 승인 시스템 (관리자 승인 후 이용 가능)

### 1-2. 프로필
- 사진, 닉네임, 핸디캡 등 프로필 관리
- 필수 정보 미입력 시 프로필 완성 유도

### 1-3. 역할 시스템
- 관리자, 방장, 운영진, 클럽운영진, 회원
- 역할별 기능 접근 권한 차등 적용

### 1-4. 회원 상태
- 회원 비활성화 (소프트 삭제, 데이터 보존)
- 활동 로그 추적

## 2. 라운딩 관리

### 2-1. 라운딩 개설
- **누구나** 라운딩 개설 가능
- 라운딩 유형: 정기모임, 컴페티션, 소셜
- 개설 시 설정: 골프장, 날짜/시간, 집합시간, 참가비(그린피/카트비/회비), 참가 마감일, 최대 인원

### 2-2. 참가 관리
- 참가 신청/취소 (회원 직접)
- 운영진이 회원을 참가 등록하는 방식도 지원
- 참가 마감일 관리
- 번호 대여 기능

### 2-3. 팀 편성
- 팀(조) 편성
- 2BB(투볼베스트볼) 팀 자동 생성

### 2-4. 게스트 초대 ⭐ 신규
- 게스트 허용 설정
- **게스트 초대링크 공유 기능**
  - 링크 접속 시 회원가입 불필요
  - 간단 정보 입력 후 해당일 스코어 입력 가능
  - 해당일 리더보드에 순위 표시
  - 게스트 데이터는 라운딩 종료 후에도 유지 (리더보드 기록 보존)

## 3. 실시간 플레이 (Play) ⭐ 안정성 최우선

### 3-1. 핵심 원칙
> **어떤 경우에도 입력된 점수가 사라져서는 안 됨**

대응해야 할 시나리오:
- 전화기 절전모드 진입
- 앱 종료
- 다른 앱으로 전환
- 배터리 방전으로 갑작스런 종료
- 네트워크 끊김

### 3-2. 저장/중단 처리
- 홀 점수 변경 시 즉시 로컬 + 서버 동시 저장
- 중간에 멈추는 경우: "저장하고 멈추기" vs "저장하지 않고 멈추기" 명확히 구분
- 복귀 시 서버/로컬 비교하여 최신 데이터 복원
- 저장 실패 시 큐에 쌓아뒀다가 재시도
- **중간에 갭(데이터 손실)이 없어야 함**

### 3-3. 기능
- 스코어 실시간 입력 (18홀)
- 팀메이트 스코어 교차 검증
- NTP(Nearest to Pin) 기록
- 자동 플레이 활성화/비활성화 (시간 기반)
- 포섬 모드 지원
- 게스트도 스코어 입력 가능 (초대링크 통해)

## 4. 회비/재정 관리 ⭐ 대폭 변경

### 4-1. 참가비 처리 (2가지 경로)

**경로 A: 회원 직접 참가**
- 참가 버튼 클릭 → 자동으로 참가비 청구
- 운영진이 회비 납부 처리 → 잔액 0

**경로 B: 운영진 등록**
- 회원이 신청 → 운영진이 참가 등록
- 회비 납부 여부 동시 처리
- 미납 시 미수금으로 해당 회원 계좌에 청구금액 표시

### 4-2. 월간 정산 시스템 ⭐ 신규
- 매월 정기모임을 기점으로 월 단위 정산
- **운영진 입출금 입력**: 모바일 UI에서 매우 쉽고 간단하게
  - 사전 설정된 수입/지출 항목 적용하여 빠른 처리
  - 항목 예시 (수입): 참가비, 회식비
  - 항목 예시 (지출): 골프장 그린피, 점심값, 음료수, 상품, 환불, 회원 크레딧
- **정산 보고서**: 자료 입력 완료 후 저장 → 정산 보고서 페이지 자동 생성
  - 앱 내 페이지로 조회/열람 (PDF 불필요)
  - 누구나 열람 가능
  - 포함 항목: 총수입, 총지출, 항목별 내역, 회원별 미수금, 이월금
- **이월**: 남은 돈은 다음 달로 이월

### 4-3. 크레딧 시스템
- 대회 상품을 모임 내 크레딧으로 지급 가능
- 크레딧 사용처:
  - 다음번 회비 납부에 사용
  - 모임에 도네이션(기여) 가능
  - 모임 내에서 캐시처럼 사용

## 10. 비기능 요구사항

### 10-1. 모바일 퍼스트
- 모든 UI는 모바일 환경을 최우선으로 설계
- 터치 친화적 (최소 터치 영역 44px)
- 노치/Safe Area 대응

### 10-2. 안정성
- 실시간 플레이 데이터 무손실 보장
- 금융 데이터(잔액, 청구) 정합성 보장
- 네트워크 불안정 시에도 데이터 보호

### 10-3. 사용성
- 운영진의 입출금 처리가 모바일에서 30초 이내 가능해야 함
- 복잡한 작업도 최소 탭으로 완료 가능하게 설계

### 10-4. 디자인 시스템 (확정)
- **테마:** 밝고 모던 (화이트 베이스 + 포인트 컬러)
